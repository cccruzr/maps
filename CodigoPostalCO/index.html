Información

Aplicación web alternativa al visor oficial, con el fin de hacer más práctica y eficiente la búsqueda del nuevo Código Postal en Colombia.

Compatible con navegadores móviles.

Se recomienda tener una versión actualizada de su navegador de preferencia en su dispositivo.


Datos oficiales del ©Código Postal de Colombia

Aplicación web para la consulta del código postal en Colombia.

INSTRUCCIONES DE USO

  • Utilice el buscador para encontrar la ubicación deseada o haga zoom manualmente hasta el nivel necesario para que se carguen los datos del código postal.
  • Ubique el mouse en la localización aproximada de su búsqueda para conocer su código postal. En móviles debe tocar la zona.
  • Haga click o toque la lupa para usar el geolocalizador: búsqueda por dirección o lugar de interés.
  • Sea específico con su búsqueda.
  • Para encontrar su localización actual haga click en el botón de geolocalización.
